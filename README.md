# MINTS Lab Presentations
 
A shared repository for storing and organizing presentations given within the MINTS Lab.
 
## Repository Structure
 
Each lab member has their own folder named after them. Inside your folder, add your presentation files following the naming convention below so they can be sorted correctly.


## Add Your Presentation
 
1. **Create your folder** (if it doesn't exist yet) — name it after yourself.
2. **Name your file** using the format:
   ```
   topic_yyyymmdd.pptx
   ```
   where `yyyymmdd` is the date of your presentation. For example:
   ```
   CrossSensitivityCorrection_20250401.pptx
   ```
